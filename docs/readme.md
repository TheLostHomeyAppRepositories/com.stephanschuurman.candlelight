# Candlelight App Documentation

Welcome to the documentation for the Candlelight Homey app!

## Installation

The Candlelight app can be easily installed via the Homey App Store:

**[Install Candlelight on Homey](https://homey.app/en-nl/app/com.stephanschuurman.candlelight/Candlelight/)**

## About

Candlelight is a Homey app that allows you to control infrared LED candles and tealights using your Homey smart home hub. The app supports various brands and models of IR-controlled candles, enabling you to integrate your ambient lighting into your home automation setup.

## Documentation Overview

This directory contains technical documentation for developers and advanced users:

- **[ir-codes.md](ir-codes.md)** - A comprehensive list of IR codes for various LED candle brands and models. Contributions of new IR codes are welcome!

- **[infrared-codes.md](infrared-codes.md)** - Technical reference for the NEC IR protocol and Homey IR implementation, including protocol specifications and useful links.

- **[ir-homey.md](ir-homey.md)** - Documentation links and resources for working with infrared signals in Homey apps.

## Contributing

If you have IR codes for candles that aren't yet supported, please submit them! You can contribute by providing either:
- Pronto hex codes
- NEC command codes
